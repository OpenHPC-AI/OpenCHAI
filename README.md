# OpenCHAI Cluster Manager Tool
The **Open CDAC HPC-AI Cluster Manager Tool (OpenCHAI)** is a unified and modular automation framework designed to simplify and accelerate the deployment, configuration, and management of HPC and AI clusters. It seamlessly integrates provisioning, centralized authentication and authorization, automation, workload scheduling, orchestration, and monitoring by leveraging industry-standard tools such as xCAT, OpenLDAP, Ansible, SLURM, Kubernetes, Nagios, Ganglia, and C-Chakshu etc.
